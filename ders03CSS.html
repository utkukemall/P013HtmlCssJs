Seçiciler

Css ile sayfalarımızdaki elementleri belirli değerlere göre seçip kontrol edebiliriz.

En Temel Seçiciler

  1. id Seçiciler : Sayfadaki elementleri id değerlerine göre seçip css uygulayabiliriz.
  2. class Seçiciler : Sayfadaki elementleri class değerlerine göre seçip css uygulayabiliriz.
  3. element Seçici : Sayfadaki elementleri element tag name değerlerine göre(div, p, h1-h6, table..) seçip css uygulayabiliriz.
