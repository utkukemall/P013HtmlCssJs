Aritmetik Operatörler

+, -, *, /, %, ++, -- sembolleri ile hesaplama işlemlerini geçekleştirebiliriz.

Atama Operatörleri

Karşılaştırma Operatörleri

== : eşit mi, === : değerleri veri tipleriyle birlikte karşılaştırır, != : eşit değilse, !== : değerler ve tipler eşit değilse, > : büyüktür, < : küçüktür, >= : büyükse veya eşitse, <= : küçük veya eşitse, ? : ternary operatörü
